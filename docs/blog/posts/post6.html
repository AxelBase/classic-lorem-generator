Building a Static, Offline-First Lorem Ipsum Generator

Published: November 30, 2025

Most online generators require an internet connection. This one doesn’t.

After your first visit, the entire tool — all 60+ classic paragraphs, generation logic, and UI — is cached by your browser. Open it on a plane, in a tunnel, or at a remote cabin: it just works.

How It’s Possible

  • Built with SvelteKit + adapter-static → 100% static HTML/CSS/JS
  • Deployed on GitHub Pages → no server, no database
  • All text stored directly in the bundle → zero API calls
  • Bootstrap 5 loaded from CDN (cached forever)

Privacy by Design

No cookies. No analytics. No tracking pixels. Nothing ever leaves your device.

Performance

  • First load: ~150 KB (gzipped)
  • Subsequent loads: instant (cached)
  • 50 long paragraphs generated in < 50 ms on mobile

This isn’t just a tool — it’s a statement: powerful web apps don’t need servers.

The future of tools is static, private, and lightning-fast — and it’s here today.
