Deploy Your Own Custom Lorem Generator on GitHub Pages

Published: December 2, 2025

Want your team’s branding? Your favorite colors? Your company name in the title?

You don’t need to ask permission. This tool is open source and designed to be forked.

Deploy in 4 Steps

  1. Fork the repository on GitHub
  2. Edit src/app.html → change title and favicon
  3. Update colors in app.css
  4. Enable GitHub Pages in repository settings

Done. Your custom version is live at yourusername.github.io/your-repo-name

Popular Customizations

  • Add your logo
  • Change fonts (e.g., Inter, Georgia, system stack)
  • Remove the Share button
  • Default to 3 paragraphs of short text
  • Add dark mode toggle
